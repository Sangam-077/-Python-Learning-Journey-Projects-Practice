## 🔁 Project 3: Recursion Practice (Google Colab)
**Concepts:** Recursion, base cases, call stack thinking

A focused notebook working through classic recursion problems:

| Problem | Description |
|---|---|
| Factorial | `n! = n * (n-1)!` |
| Fibonacci | `fab(n) = fab(n-1) + fab(n-2)` |
| List Sum | Recursive sum without loops |
| String Reversal | Reverse a string recursively |
| Palindrome Check | Combines reversal + comparison |
| Power Function | `x^n` using recursion |
